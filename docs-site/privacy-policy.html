CB Link

プライバシーポリシー

最終更新日: 2026年3月19日

1. はじめに

CB Link（以下「本サービス」）は、保護者が子供のデバイスにおける Chrome ブラウザおよび PWA の利用時間を把握するためのツールです。本プライバシーポリシーでは、本サービスが収集・使用・保存するデータについて説明します。

2. 収集するデータ

本サービスの Chrome 拡張機能は、以下のデータのみを収集します:

  • デバイスID — 拡張機能が自動生成するランダムな識別子（UUID）。個人を特定する情報ではありません
  • アプリ名 — 利用中の PWA のドメイン名（例: youtube.com）、または Chrome ブラウザの利用を示す固定文字列「chrome」
  • 利用時間 — アプリごとの利用秒数
  • タイムスタンプ — データ送信時の日時
  • 操作状態 — デバイスが操作中（active）か無操作（idle）かの状態。利用時間の正確な計測に使用します。操作内容やキー入力は一切取得しません

3. 収集しないデータ

本サービスは以下のデータを一切収集しません:

  • 閲覧した URL の全体（ドメイン名のみ使用し、パスやクエリパラメータは取得しません）
  • ページのタイトルや内容
  • フォームへの入力内容
  • パスワードやログイン情報
  • Cookie やブラウザ履歴
  • メールアドレスや氏名などの個人を特定する情報（拡張機能側）
  • 位置情報

4. データの使用目的

収集したデータは、以下の目的にのみ使用します:

  • 保護者向けモバイルアプリでの利用時間集計・表示
  • アプリごとの利用統計の生成

5. データの保存

  • データは Google Firebase（Google Cloud Platform）上に保存されます
  • 利用ログ（usageLogs）は 30日間 保存された後、自動的に削除されます
  • 日次集計データ（dailyLogs）は 84日間 保存された後、自動的に削除されます
  • データの保存・転送には HTTPS 暗号化通信を使用します

6. データの共有・第三者提供

収集したデータを第三者に販売・提供・共有することはありません。データは、ペアリング済みの保護者アカウントからのみ閲覧可能です。

7. 使用する外部サービス

  • Google Firebase — データの保存・処理・ユーザー認証に使用します。Google のプライバシーポリシーは こちら をご確認ください

8. 保護者アプリの認証

保護者アプリでは Google アカウントによるシングルサインオン（SSO）を使用します。認証には Firebase Authentication を使用し、メールアドレスと表示名のみを保存します。

9. データの削除

拡張機能をアンインストールすると、デバイス上のローカルデータ（chrome.storage）は自動的に削除されます。サーバー上のデータは、上記の保存期間（30日 / 84日）の経過後に自動削除されます。

10. お子様のプライバシー

本サービスは、保護者が子供のデバイス利用状況を把握する目的で設計されています。子供のブラウザ利用に関して収集するのは、アプリ種別と利用時間のみであり、閲覧内容やコミュニケーションの内容は一切収集しません。

11. ポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。重要な変更がある場合は、本ページにて告知します。

12. お問い合わせ

本プライバシーポリシーに関するご質問は、GitHub Issues にてお問い合わせください。
