CB Link

Chromebook 利用時間のリアルタイムレポート

CB Link とは

CB Link は、Google ファミリーリンクでは集計できない Chromebookの利用時間を補完的に監視するツールです。

  • Chrome 拡張機能がバックグラウンドで利用時間を自動計測
  • PWA はドメインでアプリ名を自動識別（YouTube, Duolingo 等）
  • 保護者はスマートフォンアプリでリアルタイムに利用状況を確認
  • アプリ別の利用時間集計・グラフ表示
  • 複数の子どもデバイス・両親それぞれのスマホに対応

システム構成

  • Chrome 拡張機能 — 子どものデバイスにインストール。利用時間を自動計測・送信
  • 保護者アプリ（Android） — 利用状況をリアルタイム閲覧。子どもデバイスの管理・OTP 発行
  • Firebase バックエンド — データ保存・集計・認証
