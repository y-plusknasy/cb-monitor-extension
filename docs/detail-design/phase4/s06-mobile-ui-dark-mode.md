# S06: モバイルアプリ UI 改善 + ダークモード — 詳細設計書

## 1. 概要

フェーズ 4 (UX 改善・配布準備) の第 2 スプリントとして、モバイルアプリの UI ブラッシュアップとダークモード対応を行う。参考UIデザインは `design-assets/Family Link風ダッシュボードデザイン/` に配置された React プロトタイプ、カラーテーマは `design-assets/dark-color-theme.txt` および `design-assets/light-color-theme.txt` に定義されたパレットに基づく。

**スコープ:**

- アプリ名「Web Usage Tracker」→「CB Link」への統一（モバイルアプリ内 UI テキスト）
- テーマシステム導入（React Context + AsyncStorage による永続化）
- ダークモード / ライトモードの切り替え機能
- 全画面・全コンポーネントのデザイン刷新（Family Link 風 Material Design）
- タブナビゲーションのアイコン・スタイル改善

**スコープ外:**

- APK ビルド・Chrome Store 公開（S07）
- バックエンド・Cloud Functions の変更
- Chrome 拡張機能の変更

## 3. 機能要件・受け入れ基準

### 3.1 テーマシステム

**要件**: ライト/ダークテーマの切り替えを設定画面から行える。選択は AsyncStorage に永続化される。

**受け入れ基準:**

- 設定画面でテーマ切り替えボタンが表示される
- テーマ変更が即座に全画面に反映される
- アプリ再起動後も選択したテーマが維持される

### 3.2 リブランディング（モバイル）

**要件**: アプリ内テキストを「CB Link」に統一する。

| 対象         | 変更内容                                               |
| ------------ | ------------------------------------------------------ |
| ログイン画面 | "Web Usage Tracker" → "CB LINK"                        |
| ログイン画面 | サブタイトルテキスト更新                               |
| 設定画面     | フッター "Web Usage Tracker v1.0.0" → "CB Link v1.0.0" |

### 3.3 UI 刷新

**要件**: 全画面を参考UIデザインに準拠した Material Design 風に刷新する。

**受け入れ基準:**

- 全画面のカードが角丸 28px のフラットデザインになっている
- ボタンが pill 形状（角丸 28px）になっている
- アプリ別行が角丸 24px のカード形式になっている
- タブバーのアイコンが絵文字から lucide-react-native の SVG アイコンに変更されている
- ダークテーマでオレンジ系アクセント、ライトテーマで青系アクセントが使われている
